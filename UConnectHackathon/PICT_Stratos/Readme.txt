Prototype Info : 
Instructions to execute the code on Windows OS:
1. Move the "Files" folder to the D drive.
2. Create Folders "Node1", "Node2", "Node3" in D drive.
3. Execute Hardcode.py.
	This file encrypts the data by using symmetric cryptography.
	Sharding process is initiated.
	A file that maps the key, file name and node number is generated.
4. Execute Decrypt.py
5. To demonstrate the frontend:
	Run :http://localhost:3000
